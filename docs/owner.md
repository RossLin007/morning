觉察笔记写作指南
什么是觉察笔记？
觉察笔记是记录当下觉察的工具，是自省的素材库。它更侧重于即时捕捉，而非深度分析。
好的觉察笔记能帮你清晰地看到想法、情绪和行为之间的联系。
觉察笔记的作用
1. 记录当下觉察： 捕捉即时的感受、想法和洞察
2. 自省素材库： 为深度反思提供丰富的素材
3. 即时捕捉： 不追求完美，重在真实记录
4. 成长轨迹： 通过定期回顾看见自己的成长
5. 模式识别： 发现重复出现的思维和行为模式
写作框架建议
1. 情境描述： 发生了什么？
2. 内在反应： 我想到了什么？感受到了什么？
3. 行为表现： 我做了什么？说了什么？
4. 觉察发现： 我看见了什么模式？有什么新的认识？
5. 行动计划： 下次遇到类似情况，我可以怎么做？

觉察笔记示例
情境： 今天晨读时，听到某位书友分享他的困惑，我内心升起了想要"指导"他的
冲动。
内在反应： 我感到一种"我知道答案"的优越感，同时也有想要帮助他的善意。但
仔细觉察，这种"指导"的冲动背后可能有我想要证明自己价值的需求。
行为表现： 我忍住了立即给建议的冲动，而是先用心倾听他的完整表达，然后分
享了我类似经历中的感受，而不是直接给出解决方案。
觉察发现： 我看见了自己习惯性想要"拯救"他人的模式，这背后可能是我对自己
价值的不确定。真正的帮助是陪伴和理解，而不是给出标准答案。
行动计划： 下次遇到类似情况，我要先停听看，问自己："我现在想要证明什
么？"然后选择先倾听，再分享感受，而不是急于给建议。
