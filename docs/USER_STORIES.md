## Epic 1: 沉浸式学习系统

### US-1.1 每日晨读
**作为** 一名渴望自我提升的用户
**我想要** 每天收到15分钟的微课程
**以便** 在碎片时间中持续学习和成长

**验收标准**:
- [x] 首页展示今日修行卡片
- [x] 课程支持 Markdown 图文混排
- [x] 提供音频导读 (TTS)
- [x] 支持环境音白噪音（雨声、森林声等）
- [x] 阅读进度自动保存

**优先级**: P0 | **状态**: 已完成

**作为** 一名学习用户
**我想要** 在课程后完成练习题

- [x] 支持选择题练习

- [x] 即时反馈正确/错误
- [x] 练习完成计入课程进度

**优先级**: P0 | **状态**: 已完成

## 附录：用户旅程剧本

### 主角：林浩
**画像**：30岁职场人，深感内耗，想读《高效能人士的七个习惯》却屡屡弃书，渴望改变但缺乏动力。

#### 阶段一：入定与启程
1. **初见** - 清晨 6:30 打开 App，被温润的 Sage Green 色调和禅意文案打动
2. **首课** - 戴上耳机，开启环境音，边听导读边阅读，遇到触动点记录灵感
3. **AI 共鸣** - 写完笔记后获得 AI 的温暖点评和引导追问
4. **完课** - 粒子特效庆祝，获得日签卡和 XP/Coins

#### 阶段二：羁绊与共生
1. **孤独** - 坚持到第3天感到枯燥，通过雷达扫描找到伙伴 Sarah
2. **共生** - 浇水互动让树苗成长，寄送明信片表达鼓励
3. **共修** - 加入 Live Room，在白噪音和番茄钟中完成学习

#### 阶段三：深潜与进化
1. **AI 教练** - 联网搜索获取真实案例和延伸阅读
2. **量化自我** - 查看五维雷达图，感受可视化成长
3. **商店消费** - 用 Coins 兑换暗夜鎏金主题

#### 阶段四：觉醒与传承
1. **危机与复活** - 断签后用补签卡挽救连胜
2. **晋升** - 升级 PRO 会员，解锁高级功能
3. **圆满** - 21天完成所有课程，获得"习惯大师"徽章
4. **新身份** - 切换为 Mentor 角色，带领新用户
